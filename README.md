# English Teacher Landing Page
- Responsive landing page for an English teacher built with HTML and CSS .

## Features
- Responsive landing page with mobile-first approach
- Fixed navigation bar with smooth scrolling
- Floating WhatsApp contact button
- Final CTA section to encourage user contact
- CSS variables for easy theming and maintenance

## Technologies Used

- HTML5
- CSS3

## How to View
- Open the login.html file in your browser to see the login screen.
